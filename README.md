# Lab7
In this lab, Logistic Regression was applied to predict whether a user clicked on an advertisement based on features such as age, income, and internet usage. The dataset used was advertising.csv.

The following steps were performed:
	•	Loaded and explored the dataset
	•	Visualized the data using Seaborn
	•	Split the data into training and testing sets
	•	Trained a Logistic Regression model
	•	Made predictions using the trained model
	•	Evaluated the model using a classification report: with accuarcy of approximately 91%
